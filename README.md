## 🇧🇷 [Português]

O **ControlFlow 2.0** é uma plataforma de elite para gestão operacional e estética automotiva, projetada para converter dados brutos em decisões estratégicas. Desenvolvido pela **Anúbis Tech**, este sistema resolve gargalos logísticos e financeiros com uma interface de alta performance e segurança de nível bancário.

### 💎 Diferenciais Estratégicos
* **Business Intelligence Dashboard**: Monitoramento em tempo real de KPIs críticos, incluindo Receita Real, Ticket Médio e Desempenho Operacional.
* **Auditoria Estratégica (Drill-down)**: Capacidade de aprofundamento de dados, permitindo gerar relatórios detalhados estilo PDF com logs de transações, taxas de cartão e lucros líquidos.
* **Smart Supply Control**: Algoritmo de baixa automatizada de insumos por serviço executado, garantindo controle rigoroso de estoque e evitando desperdícios.
* **Camada de Segurança Admin**: Acesso restrito via Firebase Auth, protegendo a integridade dos dados financeiros e operacionais.

### 🛠️ Stack Tecnológica
* **React.js & Vite**: Arquitetura moderna para máxima fluidez e tempo de resposta.
* **Firebase Ecosystem**: Realtime Database para sincronização instantânea e Authentication para segurança.
* **Tailwind CSS**: UI/UX personalizada com design minimalista "Dark Mode" de alto padrão.
* **Lucide-React**: Iconografia técnica para uma navegação intuitiva.

## 🇺🇸 [English]

**ControlFlow 2.0** is an elite management and automotive detailing platform designed to transform raw data into strategic decisions. Developed by **Anúbis Tech**, this system addresses logistical and financial bottlenecks with a high-performance interface and banking-level security.

### 💎 Strategic Value Props
* **Executive BI Dashboard**: Real-time monitoring of critical KPIs, including Real Revenue, Average Ticket, and Operational Performance.
* **Strategic Audit (Drill-down)**: Data-drilling capability, generating detailed PDF-style reports with transaction logs, credit card fees, and net profits.
* **Smart Supply Control**: Automated depletion algorithm for supplies based on executed services, ensuring strict inventory control and waste prevention.
* **Admin Security Layer**: Restricted access via Firebase Auth, protecting the integrity of financial and operational data.

### 🛠️ Tech Stack
* **React.js & Vite**: Modern architecture for maximum fluidity and response time.
* **Firebase Ecosystem**: Realtime Database for instant synchronization and Authentication for security.
* **Tailwind CSS**: Custom UI/UX with high-end "Dark Mode" minimalist design.
* **Lucide-React**: Technical iconography for intuitive navigation.
